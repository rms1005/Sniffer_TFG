1.-   MANUAL DE INSTALACIÓN

1.1      Instalación de la JVM

Para el desarrollo de la aplicación, hemos utilizado la JDK 6.0 (Java Development Kit). Para la ejecución sin embargo, basta con instalar el entorno de ejecución, esto es el JRE 6.0 (Java Runtime Environment).

Aunque Java fue diseñado para ejecutarse sobre cualquier máquina, dispone de unos mínimos requisitos hardware y software que se muestran a continuación:

 

·         Requisitos Software (en cuanto a los sistemas operativos):

o   Windows 95, Windows NT 4.0 y versiones superiores.

o   Sun Solares 2.4, 2.5, 2.5.1, 2.6 sobre SPARC y Sun Solares 2.5, 2.5.1, 2.6 sobre x86 y versiones superiores.

·          Requisitos Hardware (en cuanto al espacio en disco)

o   Una copia normal del instalador precisa de 120 MB de espacio en disco.

 

A continuación se muestran los pasos a seguir para la instalación de la Máquina Virtual de Java en los sistemas operativos Windows y Linux.

1.1.1   Instalación en Windows

En primer lugar se ha de descargar el software necesario. Para ello basta con que descarguemos la versión adecuada (en nuestro caso la J2SE 6.0) desde la página: http://java.sun.com/javase/downloads/index.jsp

 

 

Una vez en esa página, simplemente se ha de buscar la versión disponible para el sistema operativo en el cual se vaya a instalar. Se debe aceptar la licencia y posteriormente aparecerá la ventana de descarga de la Máquina Virtual de Java. La descarga se puede efectuar o bien a través del servidor FTP de Sun o bien a través de la página web. Siempre que su conexión de red le permita utilizar FTP, es recomendable que lo descargue de este modo debido a que es más fiable, seguro y rápido; de lo contrario utilice http. Seleccione el directorio en el que desea almacenar la descarga independientemente de dónde desee instalar la aplicación posteriormente.

 

El archivo de instalación para Windows es un ejecutable (.exe). Para instalarlo simplemente se deben seguir las indicaciones del programa y decidir el directorio en el cual se desea instalar la estructura de directorios del mismo. Una vez instalado, se puede borrar el archivo de instalación.

Una vez instalado, se deben configurar las variables de entorno. La variable de entorno que utiliza Java para indicarle dónde encontrar las clases de Java se denomina CLASSPATH. Es necesario añadirle las bibliotecas de clase a esta variable (directorio lib) a menos que instalara el JDK en la opción por defecto, en cuyo caso el JDK encontrará dichas bibliotecas sin problemas.

 

A su vez es necesario modificar el PATH e incluir en él el directorio bin de la JRE instalada.

 

Las variables de entorno se pueden modificar haciendo clic en Mi PC  Propiedades > Opciones Avanzadas.

 

1.1.1   Instalación en Linux

 

En primer lugar se ha de descargar el software apropiado, el cual es el J2SE 5.0 JDK como ya hemos dicho anteriormente. Se puede descargar desde http://java.sun.com/javase/downloads/index.jsp

Se debe descargar el JDK que se prefiera en la modalidad Linux self-extracting file. Una vez descargado, se deben seguir los siguientes pasos para completar su instalación:

 

1. Establecer permisos de ejecución al archivo del siguiente modo:

chmod +x jdk-1_6_0-linux-i586.bin

2. Situarnos en el directorio en el cual queramos instalar el SDK.

3. Ejecutar el archivo de instalación. Se mostrará la licencia y se habrán de aceptar algunos términos de uso.

./jdk-1_6_0-linux-i586.bin

4. La instalación de Java se habrá finalizado.

 

Como ya se ha comentado para el caso de Windows, en Linux también se han de configurar las variables de entorno. Para ello hemos de editar el archivo .bash_profile (en distribuciones Red Hat, Fedora, Conectiva, Mandrake, etc.) o el .bashrc (en SuSE) del usuario y escribir:

 

JAVA_HOME=/home/sniffer2/jdk1.5.0

PATH=$PATH:$HOME/bin:$JAVA_HOME/bin

export PATH JAVA_HOME

1.4      Instalación de la Aplicación Sniffer-I

 

En las páginas sucesivas explicaremos los pasos que se deben seguir a la hora de ejecutar nuestra aplicación de una forma sencilla.

Hay que tener en cuenta que para que funcione la aplicación se debe tener instalada la máquina virtual de Java como ya se ha explicado con anterioridad.

A su vez, es requisito de la aplicación el que los ficheros con los que trabaje la aplicación deben tener permiso de escritura, por lo cual los ficheros de prueba los ficheros de prueba deben ser copiados en el disco duro antes de ser utilizados.

 

1.4.1   Instalación en Windows

 

Para instalar la aplicación en una máquina Windows se realiza de una manera sencilla.

En primer lugar copiar la carpeta "Aplicación SnifferI" del CD en el disco duro del ordenador para un óptimo funcionamiento de la aplicación.

Hay que ir al directorio del CD Aplicación SnifferI. Dentro de ese directorio, hacer doble clic en el icono sniffer.bat

 

Figura 19  Instalación en Windows

 

Una vez pulsado se iniciará la aplicación.

 

1.4.2   Instalación en Linux

 

Para instalar la aplicación en una máquina Linux, se deben seguir los siguientes pasos:

 

En primer lugar copiar la carpeta "Aplicacion_SnifferI" del CD en el disco duro del ordenador para un óptimo funcionamiento de la aplicación.

 

1.    Abrir la consola de Linux.

2.    Situarse en el directorio del cd Aplicación SnifferI.

3.    Escribir en la consola /bin/bash sniffer.sh

4.    Se iniciará la aplicación.
